# 🌳 互動式電子家系圖與個案紀錄產生器 (Genogram & Record Generator)

這是一套專為第一線工作者打造的**現代化家系圖繪製與紀錄生成工具**。
結合了直覺的圖形化操作與強大的文字解析能力，完美解決實務上繪製複雜家庭結構（如重組家庭、多段婚姻）的痛點，並能將視覺化的家系圖一鍵轉換為專業的個案文字紀錄，大幅提升行政效率。

## ✨ 核心特色功能

### 📊 1. 智慧與自由兼具的畫布系統 (Hybrid Canvas)
* **原生世代排版**：只需輸入簡碼（如 `男女男` 或 `121`），系統即會自動運算並完美排列第一代至第三代的節點座標。
* **🧩 自由樂高擴充區**：專為「重組家庭」與「複雜關係」設計的殺手級功能！
    * **拖曳碰撞連線 (Drag & Drop Collision)**：拖曳獨立個體碰撞目標，瞬間自動產生關係連線。
    * **磁吸平行對齊 (Magnetic Snapping)**：拖曳時會自動吸附對齊伴侶的水平高度，治癒排版強迫症。
    * **擴充子代完全體**：擴充關係生出的子代享有同等權利，可自由切換婚姻狀態與繁衍下一代。
* **直覺式微調**：支援滑鼠滾輪快速切換婚姻/同住狀態、雙擊節點直接標示年齡、文字方塊直接 Enter 換行編輯。

### 📝 2. 智慧個案紀錄聯動 (Smart Record Generation)
* **看圖說故事**：系統會自動讀取畫布上的結構，翻譯成專業的敘述（例如：「案主為一般民眾... 再婚，與案妻育有1子1女，案長子已婚...」）。
* **智慧稱謂計算**：自動判斷並寫出「案長女」、「案么子」等相對稱謂。
* **前妻/前夫過濾**：自動過濾無關緊要的旁系離婚紀錄，專注呈現與案主直接相關的擴充婚姻歷程。

### 💾 3. 專為工作流打造的 UX 設計
* **雙頁籤狀態快取**：在家系圖與個案紀錄間反覆切換時，所有拖曳進度、藍色編輯模式、表單文字皆會完美保留，絕不重置。
* **隱私與便捷**：
    * 內建可自訂的「標籤庫」（身分別、教育程度、障礙證明等），並支援本地端記憶。
    * 新增「自由保存區」，可無限擴充常用短語（如：身體狀況穩定、鼓勵下床活動等），方便快速組合紀錄。
* **一鍵匯出 JPG**：自動計算畫布邊界，裁切掉多餘白邊，產出最高畫質的家系圖圖檔。
